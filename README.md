# БизнесУчёт

**MVP системы учёта доходов и расходов для малого бизнеса**

Простое, но полностью рабочее веб-приложение, разработанное как прототип для демонстрации потенциальным инвесторам или первым клиентам.  
Приложение позволяет быстро фиксировать операции, автоматически считать баланс и хранить все данные в браузере.

---

## ✨ Основные возможности

- Добавление доходов и расходов
- Выбор категории операции (настраиваемые списки)
- Автоматический расчёт:
  - Общий доход
  - Общий расход  
  - Текущий баланс
- Фильтрация операций (Все / Доходы / Расходы)
- Удаление записей
- Сохранение данных в `localStorage` (данные не теряются при перезагрузке страницы)
- Чистый и профессиональный интерфейс
